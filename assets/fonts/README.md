# Custom Fonts

Place custom fonts here for promotion image generation.

## Recommended Fonts

Download from https://fonts.google.com/:

- **Inter** - Clean, modern sans-serif
  - `Inter-Regular.ttf`
  - `Inter-Bold.ttf`

- **Outfit** - Geometric sans-serif
  - `Outfit-Regular.ttf`
  - `Outfit-Bold.ttf`

- **Roboto** - Google's versatile font
  - `Roboto-Regular.ttf`
  - `Roboto-Bold.ttf`

## How to Install

1. Go to https://fonts.google.com/
2. Search for the font (e.g., "Inter")
3. Click "Download family"
4. Extract the ZIP file
5. Copy the `.ttf` files to this folder

## Font Priority

The system tries fonts in this order:
1. Inter (if available)
2. Outfit (if available)
3. Roboto (if available)
4. System Arial (fallback)
5. Default PIL font (ultimate fallback)
